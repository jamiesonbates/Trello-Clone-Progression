# Trello-Clone-Progression

## About
This Trello Clone to-do app is built in seven different ways to show how an application differs when utilizing various technologies.

## App Versions
* [HTML](https://github.com/jamiesonbates/Trello-Clone---HTML)
* [HTML + CSS + jQuery](https://github.com/jamiesonbates/Trello-Clone---jQuery)
* [HTML + CSS + jQuery + Handlebars + Express + Knex](https://github.com/jamiesonbates/Trello-Clone---Express-Knex-HBRS)
* [HTML + CSS + jQuery + Handlebars + Express + Knex with sessions](https://github.com/jamiesonbates/Trello-Clone---Express-Knex-HBRS-Sessions)
* [HTML + CSS + Angular](https://github.com/jamiesonbates/Trello-Clone---Angular)
* [HTML + CSS + Angular + Express + Knex](https://github.com/jamiesonbates/Trello-Clone---Angular-Express-Knex)
* [HTML + CSS + Angular + Express + Knex with sessions](https://github.com/jamiesonbates/Trello-Clone---Angular-Express-Knex-Sessions)
